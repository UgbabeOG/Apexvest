# **App Name**: ApexVest

## Core Features:

- User Login & Session Management: Users can securely log in to their accounts (created by an administrator) and manage their active sessions.
- Admin User Onboarding & Lifecycle Management: Administrators can create, update, delete, and suspend investor user accounts. Users must contact an administrator for account creation.
- Admin-Managed Balance Operations: Administrators handle all user deposit and withdrawal requests. These balance management operations are currently mocked/simulated by the administration.
- Personalized Investment Dashboard: Provide a central dashboard for users to view their consolidated balances across cryptocurrency and real estate investments.
- Detailed Investment History: Display a chronological record of all past investment transactions and portfolio adjustments for a user.
- Crypto & Real Estate Watchlist: Allow users to track the performance and key metrics of specific cryptocurrencies and real estate assets they are interested in.
- AI Investment Insights Tool: A generative AI tool that provides personalized investment insights, potential strategies, and highlights risks/opportunities based on the user's portfolio and real-time market data.

## Style Guidelines:

- Primary color: A sophisticated, deep blue (#2E4A97) to convey trust and professionalism in financial dealings, paired with a light background.
- Background color: A very subtle, cool light grey-blue (#EDF0F5) to ensure clarity and calm, promoting focus on financial data.
- Accent color: A vibrant, clear sky blue (#4DBAED) to highlight interactive elements, calls to action, and key data points.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean lines and excellent readability, which is crucial for displaying financial figures and information clearly.
- Use a set of minimalist, vector-based icons that are professional and directly relevant to finance, cryptocurrency, and real estate, maintaining a consistent style.
- Implement a clean, dashboard-centric layout with clearly delineated sections for crypto and real estate, emphasizing data visibility and ease of navigation.
- Incorporate subtle, smooth animations for state changes, data loading, and interactive element feedback, ensuring a fluid user experience without being distracting.
